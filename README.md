# restaurants-booking-table

A lightweight restaurant table booking system that sends booking details directly to a Telegram bot using Netlify Serverless Functions — **no traditional backend required**.

✅ Telegram: Free, no monthly limit  
✅ Netlify: 125k function calls/month free  

---

## 📌 Overview

This project allows users to book a restaurant table from a static website.  
When a booking is submitted:

1. The form data is sent to a Netlify Serverless Function
2. The function forwards the booking details to a Telegram bot
3. The restaurant receives instant booking notifications on Telegram

This architecture eliminates the need for a live backend server or database.
